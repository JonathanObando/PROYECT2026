SENSOR DE DISTANCIA

Objetivos

Crear y poner en marcha un sistema electrónico que, por medio de un sensor ultrasónico, pueda calcular distancias y emitir una señal visual o acústica según la cercanía del objeto detectado.

Descripción de la Actividad

El presente proyecto consiste en el desarrollo de un sistema de medición de distancia utilizando un sensor ultrasónico conectado a una placa Arduino UNO. El sensor ultrasónico funciona enviando una onda sonora de alta frecuencia (ultrasonido). Cuando esta onda choca contra un objeto, rebota y regresa al sensor. El Arduino mide el tiempo que tarda en regresar la señal y, mediante una fórmula matemática basada en la velocidad del sonido, calcula la distancia en centímetros.

El sistema trabaja bajo la siguiente lógica:

• Si la distancia medida es mayor a 100 cm
• Se enciende un LED como indicador de que el objeto se encuentra lejos
• Si la distancia es menor o igual a 100 cm
• Se activa un buzzer como señal de alerta indicando proximidad.

Este tipo de sistema puede aplicarse en:

• Sistemas de alarma
• Sensores de estacionamiento.
• Automatización industrial.
• Control de nivel o proximidad en robótica.

Materiales

• Protoboard
• LEDs
• Resistencias
• Arduino
• Buzzer

Conclusiones

El proyecto permitió comprender y aplicar el funcionamiento del sensor ultrasónico como herramienta para la medición de distancias utilizando la plataforma Arduino. Se logró desarrollar un sistema capaz de detectar la proximidad de un objeto y responder de manera automática mediante señales visuales (LED) y sonoras (buzzer), cumpliendo con los objetivos planteados.
