¿Qué es una etiqueta HTML?

Las etiquetas HTML son bloques de código que indican al navegador cómo debe mostrar el contenido de una página web.

Siempre comienzan con el símbolo < y terminan con >. Entre estos símbolos va el nombre de la etiqueta.

La mayoría tienen una etiqueta de apertura y una de cierre. Por ejemplo: <p> y </p>

Estas etiquetas permiten estructurar títulos, párrafos, imágenes, tablas y mucho más dentro de una página web.
